The folder 'Working' contains a pair of bit files for the master and slave. The bit files have been generated with the timestamps euqal to 64 bits.

The folder 'Test' contains further folders with each having a pair of bit files for the master and the salve. Each of these sub-folders are with the varying timestamp precisions. The folder name - a number - indicates the timestamp bit size (eg: 16B = timestamps equal to 16 bits, 128B = timestamps equal to 128 bits).

Note: The bit files within this folder is only for v0.0.0 (i.e. master branch only).
